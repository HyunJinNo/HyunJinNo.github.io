# MySQL SELECT
## MySQL SELECT 명령어

#### 설명
SELECT 명령어는 데이터베이스에서 데이터를 조회할 때 사용하는 명령어입니다.

#### 문법
```sql
select column1, column2, ... from table_name;
```
위의 문법에서 `column1`, `column2`, `...` 는 조회하고자 하는 테이블의 속성 이름을 나타내며, `table_name`은 데이터를 조회하려고 하는 테이블의 이름을 나타냅니다.
